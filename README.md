# Tempa Nuxt Foundation

This repo is now structured for:

- `Nuxt 3`
- `GSAP`
- `@nuxtjs/i18n`
- `@nuxt/content`

## Content workflow

Write content in markdown under:

- `content/en/services`
- `content/en/case-studies`
- `content/en/blog`
- `content/id/services`
- `content/id/case-studies`
- `content/id/blog`

Collections are defined in `content.config.ts`, so Nuxt Content can query markdown files like structured records.

## Next step

Install dependencies, then run:

```bash
npm install
npm run dev
```
